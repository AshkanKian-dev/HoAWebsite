## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #8B4513;
    --secondary-color: #D2691E;
    --dark-bg: #1a1a1a;
    --light-bg: #f5f5f5;
    /* ... */
}
```

### Content
Edit the HTML content in `index.html` to match your needs.

### Functionality
Add or modify JavaScript features in `script.js`.
